Jason Hutchens: puzzles

Paul Williams and I like to waste time writing programs which solve numerical problems using brute-force techniques. They provide a challenge which has nothing to do with our work, which is only a good thing! Within these pages you will find discussions of the various puzzles, progress to date, performance details, and source code to download.

Power Summation Which numbers satisfy the following property... the sum of the individual digits of the number raised to the power of that digit is equal to the number itself?
Marriage Problem A society consists of equal numbers of males and females. Each female knows all the males and vice versa. Each female has a priority list of the males she would like to marry, and vice versa. The task is to marry them all off with the least shortfall in expectations, that is, with maximum of priorities being satisfied.

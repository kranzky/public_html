Jason Hutchens: webbot changes

This page is a record of the changes made to the WebBot system.

Version 1.5: 19/08/2000

  • Slight change so that a new stimulus does not implicitly reward the old stimulus/response. You must now explicitly reward the program for good behaviour.
  • Online documentation updated accordingly.

Version 1.4: 16/08/2000

  • Shaping changed so that WebBot is punished and rewarded automatically until the response it gives matches the target response, and no other response has a strength equal to that of the target response.
  • Added a page showing an example training session.

Version 1.3: 16/08/2000

  • Method of updating probability estimates changed, resulting in the associating strengths becoming non-zero on the very first punishment or reward, meaning that WebBot starts learning immediately.
  • Non-technical explanation of WebBot added.
  • List of version changes added.

Version 1.2: 15/08/2000

  • Strength of punishments and rewards changed, allowing for more rapid adaption.

Version 1.1: 15/08/2000

  • Name changed to WebBot.
  • URL of WebBot page changed.
  • Web pages are expanded to include more information, and greater functionality when teaching WebBot, including the ability to explicitly reward WebBot, and much better feedback.
  • The server is given a higher level of fault tolerance.
  • Each user is given their own personal WebBot to train.

Version 1.0: 14/08/2000

  • WebBot goes online.

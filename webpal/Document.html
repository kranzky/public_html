Jason Hutchens: WebPal

This is the home page of WebPal, a mock-up implementation of the envisaged commercial system which offers functionality within a web site. At the moment WebPal only works with Internet Explorer 5.5, but other versions will appear soon.

WebPal consists of:

  • an interface, based on Ofer's design and implemented in dynamic HTML;
  • a simple client, which takes user input from the interface and processes it;
  • a server, which the client communicates with to form a reply to the user's input; and
  • an API, which allows the client to perform various web navigation actions.

You may be wondering where WebPal is? To find out, type "hello".
